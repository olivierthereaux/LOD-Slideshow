A simple set of principles

  • Use URIs as names for things
  • … HTTP URIs so people can look up those names
  • Behind URIs: provide useful information, using standards
  • Include links to other URIs, so that they can discover more things

The BBC is the authority for data about our programmes

/programmes v2 - data views

  • Producing JSON, XML, RDF representations
  • Uses declarative mappings in the models
  • Ensures objects are consistently represented across feeds
  • Uses EasyRdf for RDF views

/programmes v2 - template testing

  • Old stack: controller tests are also template tests
  • Old stack: relies on CSS selectors
  • New stack: Unit tests for templates
  • Relying on extracted data (RDFa) rather than presentational elements

Testing the programmes A-Z

    public function testLetterSearch()
    {
        $this->setDefaultComponent('/components/atoz/letters');
        $data = (object) array(
            "by"      => "by",
            "search"  => "b",
            "slice" => "player",
            "letters" => array('@', 'a', 'b', 'c'),
        );

        $this->assertTriples($data, array(
            array('/programmes/a-z/by/b', 'rdfs:seeAlso', '/programmes/a-z/by/%40/player'),
            array('/programmes/a-z/by/b', 'rdfs:seeAlso', '/programmes/a-z/by/a/player'),
            array('/programmes/a-z/by/b', 'rdfs:seeAlso', '/programmes/a-z/by/c/player'),
        ));
    }
